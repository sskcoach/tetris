## 개발 과정 요약

### 초기 설정 (1-10)
- 테트리미노 개념 이해
- 게임 보드 하드코딩 및 디자인
- Screen buffer 구조 구현

### 게임 로직 구현 (11-21)
- 블록 이동 로직
- 키보드 입력 처리 (방향키 포함)
- 다음 블록 미리보기
- 블록 회전 (7가지 테트리미노)
- 라인 클리어
- 하드 드롭 기능

### 최적화 및 개선 (22-23)
- 화면 깜빡임 해결 (ANSI escape code)
- 키 입력 버퍼 처리

### 게임 시스템 (24-27)
- 스테이지 시스템
- 씬(Scene) 구조: SPLASH, TITLE, GAME, STAGE_CLEAR, GAME_OVER
- 스테이지 클리어 전환 효과
